# 网赚网盘解析WEB端

-------

## 前言

对接了公用API,支持20+种网赚网盘解析,详情查看：[博客](https://www.ilzya.com/archives/4/)

## 安装

1. clone项目 ` clone https://github.com/lzyAB/netdiskresolve.git`
2. 上传至服务器 (服务器需要支持PHP5.*版本)
3. 解压到服务器
4. 访问index.html
